## Formato padrão das respostas

**Sucesso:**
```json
{
  "success": true,
  "message": "Operação realizada com sucesso.",
  "data": {}
}
```

**Erro:**
```json
{
  "success": false,
  "message": "Mensagem do erro.",
  "error": {}
}
```
